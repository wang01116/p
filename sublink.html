📌 使用说明 1. 在左侧「Worker 地址」填入你部署的 Cloudflare Worker URL
2. 粘贴订阅链接（支持 Clash YAML / Base64）或手动输入节点 URI
3. 解析后点击节点查看详情、复制配置、导出 Clash YAML
4. 切换到「浏览器」标签，通过 Worker 中转访问外部网站
⚠️ 关于浏览器直接访问外网 浏览器无法直接建立 Shadowsocks / VMess TCP 连接。「浏览器」标签的访问方式是：请求先发到你的 Cloudflare Worker，Worker 再请求目标网站并转发响应。这不是传统意义上的代理客户端，而是 HTTPS 级别的内容中转，适合浏览网页。
